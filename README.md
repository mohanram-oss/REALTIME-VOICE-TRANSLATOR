
PRE REQUIRMENTS :


KINDLY PIP INSTALL THESE PACKAGES IN ORDER TO EXECUTE THE PROGRAM WITHOUT ANY ERRORS ,



1. pip install playsound==1.2.2

2. pip install googletrans==3.1.0a0

1. give your speech first that you have to translate
